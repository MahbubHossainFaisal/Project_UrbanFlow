## Key Concepts Learned

1. **Bronze Layer Purpose:**
   - Raw data loaded AS-IS
   - Append-only (never UPDATE/DELETE)
   - Source of truth for recovery

2. **Audit Columns:**
   - SOURCE_FILE: Track which file data came from
   - LOADED_AT: Timestamp for incremental loads

3. **Chunked Reading:**
   - Large files (50MB+) can't fit in memory
   - Use pyarrow row groups or pandas chunksize

4. **Production Code Patterns:**
   - Logging > print()
   - Retry logic for network operations
   - Return values for orchestration (Airflow needs this)
   - Connection cleanup in finally blocks
